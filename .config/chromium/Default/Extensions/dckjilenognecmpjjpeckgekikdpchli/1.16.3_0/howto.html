FAQ
  • What's the difference between lite and plus?
    SoundControl lite does not include any new plus features, but it ensures no advertising through SoundControl and you get a license key for the base features.
  • How do I know that SoundControl plus is activated on my computer?
    Just open the options page of SoundControl. The title will show that you have SoundControl plus. (You may need to wait up to five minutes for it to activate) And soon many new features will be available to you!
  • Can I use SoundControl on more than one computer?
    Yes! This SoundControl plus upgrade will work on any computer which has Chrome syncing to the same Google account.
  • Will SoundControl remain free?
    Yes. The plan is for SoundControl to remain free to use with Pandora. If you would like to use SoundControl with other web music and players, you can upgrade to plus. However, plus or not, please consider contributing to the project -- any contributions will ensure that I can continue supporting SoundControl.
  • What about privacy?
    SoundControl is focused on keeping extension permissions to a minimum and only requesting additional permissions as needed.
